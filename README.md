##Hướng dẫn cài đặt
** 1. Một số phần mềm cần cài đặt: **
- XAMPP
- PHP
- Git-bash

Chạy các lệnh sau:

```bash
composer install (Cài composer vào lcoal)
cp .env.example .env (Đổi file .env.example thành .env chính để cấu hình)
touch database/database.sqlite
php artisan key:generate
php artisan migrate
npm install 
npm run dev
```
